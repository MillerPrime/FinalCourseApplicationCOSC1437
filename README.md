# Final Course Application - COSC 1437

## What it does
This program calculates student grades and course averages.  
It allows the user to:
- Enter multiple courses and their sections
- Input student names, IDs, and grades for quizzes, exams, programs, and SLOs
- Apply extra credit to student averages
- Assign letter grades based on averages
- Determine scholarship eligibility
- Display statistics such as highest/lowest averages, students above/below grade thresholds, and total scholarships
- Print a final report showing all courses and their averages

## How to run it
1. Open the project in your Java IDE.
2. Compile the `GradeCalculator.java` file.
3. Run the program.
4. Follow the prompts in the console to enter course and student information.
5. The program will display course reports and a final summary of all courses.
